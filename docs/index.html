V3 Builds

V3 is the latest version of BFDI: BRAWL and is actively being developed. It is designed from the ground-up with bedrock foundations to ensure the game is able to grow into a fully-fledged open-source platform.

Test builds below are WebGL builds that are playable on your browser. You do not need to install them on your computer.

Want to share ideas, suggestions and feedback? Please visit the BFDI: BRAWL Discord server
